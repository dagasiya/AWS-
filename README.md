# IAMとは？

IAM（Identity and Access Management）は、AWS上で「誰に、どのサービスを、どこまで使わせるか」を制御するサービス。

## 💡 ポイント

- ユーザーには一人一つのアカウントを発行
- ロールはサービス用のアカウント（EC2など）
- ポリシーはJSON形式で権限を管理
